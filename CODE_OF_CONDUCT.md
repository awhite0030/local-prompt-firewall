# Code of Conduct

We are committed to a welcoming, harassment-free community. Be respectful, assume good intent, give actionable technical feedback, and never publish another person’s private data, credentials, or security report.

Unacceptable behaviour includes harassment, discrimination, threats, doxxing, disclosure of secrets or PII, and bad-faith disruption. Report concerns privately through the security contact in [SECURITY.md](SECURITY.md). Maintainers may remove content or participation that violates these standards.
