# Security Policy

## Supported versions

Only the latest released version is supported.

## Reporting a vulnerability

Do not create a public issue for a vulnerability, real secret, personal data, or reproduction that could expose users. Use GitHub’s private vulnerability-reporting feature for this repository once enabled, or contact the maintainer through the email listed in the repository profile.

Include a minimal synthetic reproduction, affected version, impact, and suggested mitigation. We will acknowledge reports within 7 days and coordinate disclosure after a fix is available.
