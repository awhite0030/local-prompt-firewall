# Local Prompt Firewall — кратко по-русски

Расширение проверяет текст перед отправкой в ChatGPT и предлагает заменить найденные чувствительные значения локальными токенами. Исходный промпт, найденные данные и таблица подстановок не отправляются на сервер разработчика: в проекте нет бекенда, аналитики и внешнего API.

## Установка

```bash
npm ci
npm run build
```

Откройте `chrome://extensions`, включите режим разработчика и выберите **Load unpacked** для папки `apps/chrome-extension/dist`.

## Ограничения

Первая версия проверяет только текстовое поле ChatGPT. Она не анализирует вложения, изображения, голос, canvas и не гарантирует обнаружение всех персональных данных. Перед отправкой всегда проверяйте текст самостоятельно.

Подробности: [модель угроз](THREAT_MODEL.md) и [политика приватности](PRIVACY.md).
